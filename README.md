# HSK Chinese Learning Web App

An interactive web application for learning Mandarin Chinese vocabulary, designed for HSK (Hanyu Shuiping Kaoshi) exam preparation with English and Mongolian translations.

## Live Demo

[**Try it here**](https://bbayalagmaa.github.io/hsk-chinese-learning/hsk1-learning.html)

## Features

### Learn & Write
- **Vocabulary Cards** - Chinese characters with pinyin, English, and Mongolian translations
- **Stroke Order Practice** - Interactive stroke animation using HanziWriter library
- **Multi-character Support** - See stroke order for all characters in compound words
- **Audio Pronunciation** - Click to hear native pronunciation (Text-to-Speech)
- **Example Sentences** - Real usage examples for each word

### Story Mode
- **Reading Practice** - Short stories using HSK vocabulary
- **Ruby Pinyin** - Pinyin displayed above each character
- **Toggle Pinyin** - Show/hide pinyin for self-testing
- **Bilingual Translation** - English and Mongolian translations

### Quiz Mode
- **10 Random Questions** - Test your knowledge with multiple choice
- **Instant Feedback** - See correct answers immediately
- **Score Tracking** - Track your quiz performance

### Exam Mode
- **20 Questions** - Comprehensive level assessment
- **Pass Requirement** - Score 80% or higher to unlock next HSK level
- **Level Progression** - Unlock HSK2, HSK3, HSK4 by passing exams

### Daily Slang
- **100 Slang Words** - Learn casual Chinese expressions
- **Learn & Track** - Mark words as learned
- **Personal Bank** - Review all learned slang words

### Progress Tracking
- **Word Progress** - Track learned words per level
- **Persistent Storage** - Progress saved in browser (localStorage)
- **Visual Progress Bar** - See your completion percentage

## HSK Levels

| Level | Words | Description |
|-------|-------|-------------|
| HSK1 | 150 | Basic vocabulary - greetings, numbers, family, daily life |
| HSK2 | 15 | Elementary - conjunctions, common verbs, adjectives |
| HSK3 | 15 | Intermediate - abstract concepts, work-related terms |
| HSK4 | 15 | Upper-intermediate - formal expressions, advanced grammar |

## How to Use

1. **Select Level** - Click HSK1, HSK2, HSK3, or HSK4 button
2. **Learn Words** - Use arrow buttons to navigate vocabulary cards
3. **Practice Strokes** - Click "Play All" to see stroke animations, "Quiz" to practice writing
4. **Read Stories** - Switch to Story tab to read with pinyin
5. **Take Quizzes** - Test yourself with Quiz mode
6. **Pass Exams** - Score 80%+ to unlock the next level
7. **Learn Slang** - Check out daily slang words at the bottom

## Technologies

- **HTML5** - Single-file web application
- **CSS3** - Responsive design with gradients and animations
- **JavaScript** - Interactive functionality
- **HanziWriter** - Chinese character stroke animations
- **Web Speech API** - Text-to-speech pronunciation
- **localStorage** - Progress persistence
